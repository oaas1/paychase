# PayChase

Invoicing dashboard for Nigerian freelancers. Send invoices via WhatsApp. Get paid faster.

## Features
- Dashboard with stats (total invoices, pending payments, paid this month, overdue)
- Create new invoices
- (Coming soon: WhatsApp integration, payment links)

## Tech Stack
- HTML5
- CSS3
- JavaScript (coming soon)
- Node.js + Express (coming soon)
- PostgreSQL (coming soon)
